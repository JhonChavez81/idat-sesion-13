El Detrás de Escena de la Web

Fundamentos para la creación de un sitio web (Bloque 1)

  • Lo que vemos vs. Lo que es: La web no es magia, es una estructura ordenada.
  • Los Cimientos: Hoy entenderemos cómo funciona todo antes de escribir código.
  • El Objetivo: Construir sitios web que no solo se vean bien, sino que funcionen perfecto.

¿Cómo nos conectamos realmente?

El viaje de la información

1. Internet

Es la enorme autopista mundial de cables submarinos y satélites que conecta todas las computadoras.

2. Navegador

Es tu "vehículo" y traductor. Toma el código aburrido de la autopista y lo convierte en diseño visual.

3. Cliente / Servidor

La regla de oro: Tú (el cliente) pides ver una página, y la otra computadora (el servidor) te la entrega.

La Trilogía del Alojamiento Web

La analogía de la casa

1. El Dominio

(La Dirección)
Es el nombre de tu web (ej. idat.edu.pe). Sin esto, la gente no sabría cómo encontrar tu casa en el mapa.

2. El Hosting

(El Terreno)
Es el espacio virtual que alquilas para guardar tus archivos, imágenes y el código HTML de tu web.

3. El Servidor

(La Computadora)
La super-máquina que nunca se apaga y muestra tu "casa" a los visitantes las 24 horas del día.

Trazando la Ruta: Arquitectura Web

Si el usuario no lo encuentra, no existe.

El Mapa de Sitio (Sitemap)

Es el esqueleto o índice de nuestro proyecto. Nos ayuda a organizar jerárquicamente las páginas antes de programar (Inicio > Servicios > Contacto).

Regla de oro: Todo debe estar a un máximo de 3 clics de distancia.

El Menú: Expectativa vs Realidad

El Mapa de Sitio se transforma en tu Menú de Navegación

Sin Mapa de Sitio (Caos)
Inicio Nuestra Empresa Misión Visión Servicios Web Apps Contacto Lima Contacto Piura

Demasiadas opciones. El usuario se abruma, no sabe dónde hacer clic y abandona la página.

Con Mapa de Sitio (Ordenado)
Inicio Nosotros Servicios Contacto

Limpio y claro. Los servicios específicos (Web, Apps) se agrupan elegantemente dentro de un menú desplegable.

Vistiendo la Estructura

Elementos gráficos para la web

Imágenes

Deben ser ligeras y optimizadas para que la web cargue rápido.

Tipografía

Fuentes legibles (como las de Google Fonts) para no cansar la vista.

Color

Paletas que respeten la identidad de marca y guíen la atención.

La Trinidad de la Web

Los 3 lenguajes que hacen posible todo

HTML5

El Esqueleto 💀
Define la estructura, los textos, imágenes y enlaces. Sin HTML no hay página.

CSS3

La Ropa / Piel 👕
Añade diseño, colores, tamaños y posiciones. Hace que el esqueleto se vea hermoso.

JavaScript

Los Músculos 💪
Le da interactividad, movimiento y lógica. Hace que la página "piense" y reaccione.
